# Công cụ Tách & Gộp PDF

Công cụ web miễn phí để tách và gộp file PDF, sắp xếp lại trang, xem trước trước khi tải về.

## Tính năng

- ✅ Gộp nhiều file PDF thành một
✅ Gộp thành sổ (2-up): Tự động dàn trang bìa và nội dung để in ấn chuyên nghiệp
✅ Tách file PDF thành từng trang riêng
✅ Xem trước các trang PDF
✅ Sắp xếp lại thứ tự trang bằng kéo thả
✅ Chọn trang muốn giữ lại và gán vai trò (Mặt đầu, Nội dung, Mặt cuối)
✅ Tự động đồng nhất kích thước trang khi gộp
✅ Đặt tên file khi tải về
✅ Giao diện đẹp, hiệu ứng mượt mà
✅ Hoạt động hoàn toàn trên trình duyệt (không upload lên server)

Cách sử dụng

1. Mở trang web
2. Kéo thả hoặc chọn file PDF
3. Chọn chức năng:
   - **Gộp File**: Gộp tất cả file thành một theo thứ tự.
   - **Gộp Thành Sổ**: Chế độ dàn trang in ấn. Bạn có thể chọn vai trò (Bìa, Nội dung) cho từng trang để in ra đóng thành sổ.
   - **Tách File**: Tách từng trang thành file riêng.
   - **Xem Trước**: Xem, chọn, sắp xếp lại trang và gán vai trò gộp sổ trước khi tải về.

## Bảo mật

Tất cả xử lý được thực hiện trên trình duyệt của bạn. File không được upload lên server nào.
